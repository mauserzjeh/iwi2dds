# iwi2dds
A small CLI tool to convert IWI (Infinity Ward Image) files to DDS (Direct Draw Surface) files


# Build from source
```
go build -o iwi2dds.exe
```

# Usage

Help
```
iwi2dds.exe -help
```

Convert
```
iwi2dds.exe -i "path/to/iwi/file.iwi" -o "path/to/dds/file.dds"
